# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-04-29

Initial public release.

### Added

- `bag2rawsync.py` for ROS bag raw-data extraction
- all-topic export support
- image, Livox lidar, IMU, and NavSatFix dedicated exporters
- JSONL export for custom topics
- same-timeline alignment mode with configurable anchor topic
- overwrite protection and optional serialized payload export
- English and Chinese documentation
- repository cleanup with root `requirements.txt` and `.gitignore`
